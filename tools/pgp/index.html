There is no PGP software to rule them all. There are several PGP clients that you would need for all platforms. Below is a list of tools for all platforms with their download links.

There is no standalone PGP app for IOS as of now. Check out ProtonMail.

GPG4Win (GnuPG)

The GnuPrivacyGuard or GPG is the open source version of PGP. Both mean the same thing. Windows calls it Gpg4Win. You don't want the commercial version of PGP, because it is not open source, which means a backdoor can be hidden in the source code and nobody will know.

GPG has two versions: the full version, which includes command line support with the gpg command, and a GUI, called Kleopatra. The light version has no GUI, only command line. To get a list of commands for GPG on the command line, run gpg --help.

GPGTools

GPGTools is the PGP software for OSX machines. It is actually a collection of four tools - MacGPG is the underlying componant of GPGTools. You can make use of the command line there, GPG Keychain for a GUI key management interface, GPG Services to bridge into other applications, and GPG for Mail, a plugin for Mail that allows for easy encryption with a click.

  • desktop_macDownload for Mac

  • codeSource Code

OpenKeychain

OpenKeychain is PGP for Android. It started as a fork of APG when development stopped. APG did merge what OpenKeychain did back in 2014, but there has not been an update as of this writing (2016). It wraps everything in an easy to use GUI - no command line. It allows for signing and/or encryption of text/files, key management, decryption of text/files, fingerprint verification, and more. The output for text is copy to clipboard, or share directly to another app; file encryption is save to disk or share to another app.

ProtonMail

ProtonMail is an encrypted email provider, hosted in Switzerland, enjoying the best privacy protections in the world. PGP is employed for encryption, and will auto-encrypt when sending to @protonmail.com or @protonmail.ch address. Two passwords are used: the first one proves your identity and the second one unlocks your private key. The password for the private key never leaves your computer. You can encrypt email with a passphrase (symmetric encryption) to send to non ProtonMail addresses where the recipiant can reply up to five times, in which they would have to create an account to continue talking on that email. You can also set a message to self-destruct after a set time.
