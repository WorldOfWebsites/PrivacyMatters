PGP

There is no PGP software to rule them all. There are several PGP clients that you would need for all platforms. Below is a list of tools for all platforms with their download links.

GnuPrivacyGuard (GPG or GnuPG)

The GnuPrivacyGuard or GPG is the open source version of PGP. Both mean the same thing. Windows calls it Gpg4Win. You don't want the commercial version of PGP, because it is not open source, which means a backdoor can be hidden in the source code and nobody will know.

GPG has two versions: the full version, which includes command line support with the gpg command, and a GUI, called Kleopatra. The light version has no GUI, only command line. To get a list of commands for GPG on the command line, run gpg --help.
