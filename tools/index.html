Criteria

The criteria for an app to be considered secure and therefore safe to use is as follows:

  • The app must be open source.
  • The app must be end-to-end encrypted and the server must never handle the user's private keys at anytime. The server by extension should never see plaintext. Only ciphertext.
  • No home-brewed crypto ever. These have not been audited by experts as much as the time-tested algorithms like AES.
