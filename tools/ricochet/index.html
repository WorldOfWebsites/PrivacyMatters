What is Ricochet?

WARNING: Ricochet is experimental and its protocol has not been audited. Don't use this if your life is in danger. Above all, use at your own risk.

Ricochet allows for end to end encrypted and anonymous communicaions. It uses Tor and Tor Hidden Services as its main building blocks. Your Ricochet ID is also the address of the hidden service as it uses hidden services as endpoints. This is ideal since connections never leave the Tor network, so the connection is encrypted. Hidden services are also ideal because you cannot learn the IP address of the hidden service. There are no rlay servers, you are the servers. It's really two server talking to eachother. Metadata is eliminated. A ricochet ID looks like ricochet:some_random_characters. You can send contacts a request and if they accept, you can chat with them. The developers have said that file sending will be coming soon.
