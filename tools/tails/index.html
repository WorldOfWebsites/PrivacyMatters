Tails is a secure operating system that sends all traffic through the Tor network. It is installed and run off of a USB or DVD and is loaded onto RAM. The host computer's disk does not get used. If you need to stay anonymous, this is the OS you use. Edward Snowden and the journalists that worked wth him used it, for example.

Tails is based off of Debian and includes software such as:

  • Tor Browser
  • GIMP
  • Pidgin
  • Icedove
  • GPG

A full list of included software can be found here.

Tails is full of counter-surveillance software. When you shutdown Tails, or physically remove the boot device, it will wipe the RAM so that sensitive data cannot be extracted. You can also wipe individual files and/or free space on the system as well and it has a virtual keyboard you can use if you suspect there is a keylogger.

You can get Tails from

  1. The internet as an ISO
  2. From a friend who has Tails (make sure they've properly verified it!)

If you're installing from Debian, Ubuntu or Mine, your life just got a whole lot easier. You can get the Tails installer which will install Tails on your media of choice. Tails systems installed with the Tails Installer benefit from updates that don't require you to clone another ISO. You will also not need to have 2 USB sticks.

Encrypted Persistant Storage

You can create an encrypted persistant storage so you can store data that will remain on each boot. Anything not saved in the Persistant folder is lost when Tails is shutdown. You can configure it by goint to Applications > Tails > Configure Persistant Storage. It will ask you to provide a passphrase and to choose what to store. Personal Data is already enabled and you can choose to activate more options as needed. Everytime you make a change to what the persistant volume saves, you must restart Tails or that anonymous IM account you wanted to save will disappear.

Tails Won't Boot

Sometimes, Tails won't boot off the computer. You will need to access your BIOS or UEFI settings. Once you've went there, try:

  • Changing the boot order so the computer checks for a bootable USB or DVD first.
  • Turning off secure boot. (UEFI devices only, pretty much anything that shipped with Windows 8 out of the box) Tails does not have a secure boot key like Windows or Ubuntu does. This does make the computer more at risk, so try this last.
  • Disable fast boot.
  • Try to start on Legacy BIOS or if already on that, try UEFI if you have it.

More in depth details can be found in the installation assistant (at the next card).
