Signal Private Messenger is an easy to use messaging app by Open Whisper Systems. It employs end-to-end encryption for Signal contacts, and plaintext sms/mms. The app uses the Signal Protocol, also developed by Open Whisper Systems. The messages are encrypted at rest if you enable a passphrase on Android, and always on IOS. If sending a message to another Signal user, the message is encrypted on the device and sent to the server. Metadata is secured over HTTPS. The server never has access to any private keys at any point. All the code is open source and can be found on GitHub, which means you can go over the code to make sure nothing fishy is going on. If you don't trust the app store apk's, you can build from the source code (on Android at least).

The Signal Protocol is regarded as the gold standard in encrypted messaging, enough to be adopted by WhatsApp, Google Allo and Facebook Messenger. The Signal Protocol allows for:

  • Forward secrecy: past messages cannot be decrypted, and the key used to encrypted messages is changed constantly.
  • Identity verification
  • Designed to work in an asyncronous environment, meaning it recognizes somebody might be offline.
  • Disappearing messages

Registering is easy - all you need is a phone number where you can receive a text message or phone call.

Mobile

  • Select your country and enter your phone number. A phone number is your username. It allows other peple who have your phone number in their contacts to automatically sent encrypted messages to you.
  • A text message with a verification code will arrive. If you are using Android, the code is automatically detected and entered. IOS users must enter the code themselves. If the message does not come through, or are registering a home phone's number, once the verification has timed out, press "call me" and the code will be spoken and you will enter the code.
  • The app will proceed to generate keys.
  • Signal will ask if you want to import your messages into its encrypted database. Tap on the message to say yes.
  • Congratulations! You have now set up Signal. Happy chatting! You may wish to delete your messages from the old app since Signal uses a message database other than the database Android provides.

Desktop

  • Once you have installed the desktop app (link below), launch it and click "Get Started".
  • Android: Open Settings > Devices > Linked Devices. Click the + to open the scanner.
    IOS: Open Settings > Linked > Devices > Link New Device to open the scanner
  • On the desktop, give the device a name so you can identify and remove devices should the need arise.
  • The app will generate keys and begin syncing your contacts.

The app will only sync future messages, since there was no desktop key to encrypt to, and the forward secrecy ensures that the keys to decrypt old messages do not exist anymore.

Disappearing Messages

Signal allows disappering messages to be sent, ranging from 5 seconds to 1 week. It is more for keeping a tidy message history. If your contact is your advesary, they can always modify the client to remove this function or screenshot or take a picture with another camera, so keep that in mind. Here's how to use disappearing messages.

  • Android/Desktop: Tap the 3 dots on the top right and choose "Disappearing Messages".
    IOS: Long-press on your contact's name to open converation settings. Flip the switch for "Disappearing Messages" on.
  • Android/Desktop: Looks through the list for your desired timer. Tap on OK to apply. On desktop, it applies 1 day as the timer by default.
    IOS: Drag the slider to choose your timer. It will show what you have chosen above the slider. Go back to the conversation view when done.
  • The conversation view will show an update message that says "You/person set disappearing messages to n seconds/minute(s)/hour(s)/week.
    A clock icon will appear when disappearing messages are active in the top bar.
    There will be an hourglass on each message, the more it empties, the less time you have to read it.

When someone activates disappearing messages, it affects the other person or people in the group. They'll take on the same expiration timer you have set, and you will be affected by the timers they set as well.

Censorship Circumvention

Recently (late December 2016), Signal has seen efforts by Egypt, among others to censor access to Signal. In response, Signal deployed censorship circumvention to phone numbers affected by censorship by means of domain fronting which makes traffic look like, in this case a regular TCP connection to Google, where in reality, is communicating with Signal servers.

Currently, censorship circumvention is deployed in:

  • Egypt (+20)
  • UAE (+971)
  • Cuba (+53)
  • Oman (+968)

Signal plans in future releases to automatically detect censorship and deploy circumvention when needed.

If you want Signal to be truly private, you will need to change some settings. The Intercept has a great article on that subject.
